# Telegram Weather Bot

## Description

Telegram bot that provides current weather using OpenWeather API.

## Technologies

- Python
- OpenWeather API
- python-telegram-bot
- requests
- python-dotenv

## Setup

1. Create `.env` file with:
    - OPENWEATHER_API_KEY
    - TELEGRAM_BOT_TOKEN
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
